# AI Search Algorithms in Python

This repository contains implementations of various search algorithms used in Artificial Intelligence.

## 📌 Algorithms Included

### 🔹 Uninformed Search
- Breadth First Search (BFS)
- Depth First Search (DFS)

### 🔹 Informed Search
- Greedy Best First Search
- A* Search

### 🔹 Local Search
- Hill Climbing
- Steepest Ascent Hill Climbing
- Stochastic Hill Climbing
- Local Beam Search
